# GiphyPWA

## Installation (local/prod)

```sh
yarn
```

## Run (local/prod)

```sh
yarn start
```

## Development (local)

```sh
yarn serve # with hot reload
```

## Run linter (test)

```sh
yarn lint
```

## Fix linter errors (test)

```sh
yarn cs-fix
```
